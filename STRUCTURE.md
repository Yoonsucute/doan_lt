# Cau truc du an

- `auth/`: dang nhap, dang ky, dang xuat, profile.
- `projects/`: danh muc, chi tiet, upload, sua, xoa, tim kiem, download source.
- `shop/`: gio hang, checkout, lich su don hang, xu ly gio hang.
- `actions/`: bookmark, follow, report, rating, comment, vote comment.
- `admin/`: dashboard, quan ly user, duyet bai, quan ly don hang.
- `includes/`: header, navbar, footer dung chung.
- `assets/`: CSS, JavaScript.
- `uploads/`: anh preview va file source upload.
- `chatbot/`: widget va API chatbot.
- `database/`: file SQL khoi tao database.

Các file PHP ở thư mục gốc như `login.php`, `cart.php`, `project_detail.php` chỉ là wrapper để link cũ vẫn chạy. Code thật nằm trong các thư mục module phía trên.
